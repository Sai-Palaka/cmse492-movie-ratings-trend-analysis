# Movie Ratings Trend Analysis

## Project Overview
This project analyzes trends in movie ratings over time to understand how different factors influence ratings and popularity.

## Objectives
- Explore trends in movie ratings over time
- Identify patterns in high-rated vs low-rated movies
- Analyze relationships between features such as genre and release year
- Perform exploratory data analysis (EDA)
- Visualize insights using graphs and charts

## Dataset
The dataset includes movie-related information such as:
- Title
- Rating
- Release year
- Genre (if available)
- Other relevant features

## Tools & Technologies
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## Project Structure
- data/ → raw datasets  
- notebooks/ → analysis notebooks  
- src/ → scripts or helper code  
- results/ → graphs and outputs  

## How to Run
1. Open the notebook in the notebooks/ folder
2. Run all cells
3. Make sure required libraries are installed:
   - pandas
   - matplotlib
   - seaborn
